# Visit with Us - Wellness Tourism Package Dataset

This dataset is used for the Visit with Us MLOps project.

## Business Objective

Predict whether a customer is likely to purchase the newly introduced Wellness Tourism Package before the sales team contacts them.

## Target Variable

`ProdTaken`

- 0: Customer did not purchase the package
- 1: Customer purchased the package

## Raw File

- `tourism.csv`

## Prepared Files

After the data preparation stage, the repository also contains:

- `train.csv`
- `test.csv`
- `data_schema.json`
